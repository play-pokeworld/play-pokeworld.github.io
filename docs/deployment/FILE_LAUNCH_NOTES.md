# Launch notes

## Recommended deployment

Deploy the contents of `dist/` to GitHub Pages.

`dist/index.html` uses a hashed Vite ES-module bundle and is the production target.

## Direct local opening

The source-root `index.html` can also be opened directly with `file://` for quick local checks. It uses classic fallback scripts instead of an ES-module entry so browsers do not block `/src/main.js` with file-origin CORS errors.

For development with hot reload, use:

```bash
npm install
npm run dev
```

Then open the local HTTP URL printed by Vite.

## Do not open `index.vite.html`

`index.vite.html` is a temporary build input generated by `npm run build`. It is removed automatically after the build and is not meant to be opened directly. Use the root `index.html` for direct `file://` launch, or `dist/index.html` for production/GitHub Pages.
